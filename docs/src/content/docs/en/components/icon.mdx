Display icons from the Lucide icon library with customizable size and color.

## Basic Usage

<div className="flex items-center gap-4 my-4 not-prose">
  <Icon name="home" />
  <Icon name="settings" />
  <Icon name="user" />
  <Icon name="search" />
  <Icon name="menu" />
</div>

```mdx
<Icon name="home" />
<Icon name="settings" />
<Icon name="user" />
<Icon name="search" />
<Icon name="menu" />
```

## Sizes

<div className="flex items-center gap-4 my-4 not-prose">
  <Icon name="star" size="sm" />
  <Icon name="star" size="md" />
  <Icon name="star" size="lg" />
  <Icon name="star" size="xl" />
  <Icon name="star" size={48} />
</div>

```mdx
<Icon name="star" size="sm" />   // 16px
<Icon name="star" size="md" />   // 20px
<Icon name="star" size="lg" />   // 24px
<Icon name="star" size="xl" />   // 32px
<Icon name="star" size={48} />   // 48px
```

## Pre-built Icons

Common status icons are available as named components:

  <CheckIcon /> Check
  <XIcon /> Error
  <InfoIcon /> Info
  <WarningIcon /> Warning
</div>


## Props

<ParamFieldGroup>
  <ParamField name="name" type="string" required>
    Lucide icon name in kebab-case (e.g., "chevron-right", "home")
  </ParamField>
  <ParamField name="size" type="'sm' | 'md' | 'lg' | 'xl' | number" default="'md'">
    Icon size (sm=16px, md=20px, lg=24px, xl=32px, or custom number)
  </ParamField>
  <ParamField name="color" type="string">
    CSS color value (e.g., "#ef4444", "red")
  </ParamField>
  <ParamField name="className" type="string">
    Additional CSS classes
  </ParamField>
</ParamFieldGroup>

Browse all available icons at [lucide.dev/icons](https://lucide.dev/icons).
