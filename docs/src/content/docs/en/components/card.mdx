## Props

### Card

<ParamFieldGroup>
  <ParamField name="title" type="string" required>
    Card title text
  </ParamField>
  <ParamField name="icon" type="string">
    Emoji icon displayed before the title
  </ParamField>
  <ParamField name="href" type="string">
    Link URL (makes the entire card clickable)
  </ParamField>
</ParamFieldGroup>

### CardGroup

<ParamFieldGroup>
  <ParamField name="cols" type="1 | 2 | 3 | 4" default="2">
    Number of columns in the grid layout
  </ParamField>
</ParamFieldGroup>
