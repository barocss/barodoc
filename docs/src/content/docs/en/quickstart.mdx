Create a new documentation site with Barodoc.

## Prerequisites

- Node.js 20 or higher
- pnpm (recommended)

## Create a New Project

```bash
pnpm create barodoc my-docs
```

The CLI will prompt you for:
- Project name
- Site name
- Primary color
- i18n support

## Project Structure

After creation, your project will look like:

```
my-docs/
├── src/
│   └── content/
│       └── docs/
│           └── en/
│               ├── introduction.mdx
│               └── quickstart.mdx
├── public/
│   └── logo.svg
├── barodoc.config.json
├── astro.config.mjs
└── package.json
```

## Development

```bash
cd my-docs
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

This generates a static site in the `dist/` directory.

## Deploy

Deploy the `dist/` folder to any static hosting:
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages
