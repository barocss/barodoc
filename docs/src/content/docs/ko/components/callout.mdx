Callout은 문서에서 중요한 정보를 강조합니다.

## 유형

### Info

<Callout type="info">
  정보 callout입니다. 일반적인 정보와 팁에 사용하세요.
</Callout>

```mdx
<Callout type="info">
  정보 callout입니다.
</Callout>
```

### Warning

<Callout type="warning">
  경고 callout입니다. 중요한 알림에 사용하세요.
</Callout>

```mdx
<Callout type="warning">
  경고 callout입니다.
</Callout>
```

### Tip

<Callout type="tip">
  팁 callout입니다. 유용한 팁과 모범 사례에 사용하세요.
</Callout>

```mdx
<Callout type="tip">
  팁 callout입니다.
</Callout>
```

### Danger

<Callout type="danger">
  위험 callout입니다. 중요한 경고에 사용하세요.
</Callout>

```mdx
<Callout type="danger">
  위험 callout입니다.
</Callout>
```

## 커스텀 제목

<Callout type="info" title="커스텀 제목">
  모든 callout 유형에 커스텀 제목을 지정할 수 있습니다.
</Callout>

```mdx
<Callout type="info" title="커스텀 제목">
  커스텀 제목이 있는 내용.
</Callout>
```

## Props

<ParamFieldGroup>
  <ParamField name="type" type="'info' | 'warning' | 'tip' | 'danger' | 'note'" default="'info'">
    아이콘과 색상 스킴을 결정하는 Callout 유형
  </ParamField>
  <ParamField name="title" type="string">
    커스텀 제목 (기본값은 type 이름)
  </ParamField>
</ParamFieldGroup>
