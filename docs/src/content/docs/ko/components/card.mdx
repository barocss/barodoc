Card는 관련 콘텐츠나 네비게이션 링크를 표시합니다.

## 기본 Card

<Card title="시작하기" icon="🚀">
  프로젝트에 Barodoc을 설정하는 방법을 알아보세요.
</Card>

```mdx
<Card title="시작하기" icon="🚀">
  프로젝트에 Barodoc을 설정하는 방법을 알아보세요.
</Card>
```

## 링크가 있는 Card

<Card title="문서" icon="📚" href="/docs/introduction">
  전체 문서를 읽어보세요.
</Card>

```mdx
<Card title="문서" icon="📚" href="/docs/introduction">
```

## Card Group

여러 카드를 반응형 그리드로 그룹화:

<CardGroup cols={2}>
  <Card title="설치" icon="📦" href="/docs/guides/installation">
    프로젝트에 Barodoc 설치하기.
  </Card>
  <Card title="설정" icon="⚙️" href="/docs/guides/configuration">
    문서 사이트 설정하기.
  </Card>
  <Card title="컴포넌트" icon="🧩" href="/docs/components/callout">
    사용 가능한 컴포넌트 살펴보기.
  </Card>
  <Card title="배포" icon="🚀" href="/docs/guides/installation">
    GitHub Pages에 배포하기.
  </Card>
</CardGroup>

```mdx
<CardGroup cols={2}>
  <Card title="설치" icon="📦" href="/docs/guides/installation">
    프로젝트에 Barodoc 설치하기.
  </Card>
  <Card title="설정" icon="⚙️" href="/docs/guides/configuration">
    문서 사이트 설정하기.
  </Card>
</CardGroup>
```

## Props

### Card

<ParamFieldGroup>
  <ParamField name="title" type="string" required>
    카드 제목 텍스트
  </ParamField>
  <ParamField name="icon" type="string">
    제목 앞에 표시되는 이모지 아이콘
  </ParamField>
  <ParamField name="href" type="string">
    링크 URL (전체 카드를 클릭 가능하게 만듦)
  </ParamField>
</ParamFieldGroup>

### CardGroup

<ParamFieldGroup>
  <ParamField name="cols" type="1 | 2 | 3 | 4" default="2">
    그리드 레이아웃의 열 수
  </ParamField>
</ParamFieldGroup>
