## 다중 열기

`type="multiple"`로 여러 항목을 동시에 열 수 있습니다:

<SimpleAccordion client:load type="multiple" items={[
  { title: "프론트엔드", content: "React, Vue, Svelte 등이 지원됩니다." },
  { title: "백엔드", content: "Node.js, Python, Go 및 기타 백엔드가 원활하게 작동합니다." },
  { title: "데이터베이스", content: "PostgreSQL, MySQL, MongoDB 등이 문서화되어 있습니다." }
]} />

```mdx
<SimpleAccordion client:load type="multiple" items={[
  { title: "프론트엔드", content: "React, Vue, Svelte..." },
  { title: "백엔드", content: "Node.js, Python, Go..." },
  { title: "데이터베이스", content: "PostgreSQL, MySQL, MongoDB..." }
]} />
```

## 기본 열림

`defaultValue`로 초기에 열린 항목을 설정합니다:

<SimpleAccordion client:load defaultValue="item-1" items={[
  { title: "첫 번째 항목", content: "기본적으로 닫혀 있습니다." },
  { title: "두 번째 항목", content: "기본적으로 열려 있습니다." },
  { title: "세 번째 항목", content: "기본적으로 닫혀 있습니다." }
]} />

## Props

<ParamFieldGroup>
  <ParamField name="items" type="AccordionItem[]" required>
    아코디언 항목 배열 (아래 Accordion Item 참조)
  </ParamField>
  <ParamField name="type" type="'single' | 'multiple'" default="'single'">
    하나 또는 여러 항목을 동시에 열 수 있는지 여부
  </ParamField>
  <ParamField name="collapsible" type="boolean" default="true">
    type이 "single"일 때 모든 항목을 닫을 수 있는지 여부
  </ParamField>
  <ParamField name="defaultValue" type="string | string[]">
    초기에 열린 항목 - "item-0", "item-1" 등 사용
  </ParamField>
  <ParamField name="client:load" type="directive" required>
    클라이언트 측 상호작용을 위한 필수 Astro 디렉티브
  </ParamField>
</ParamFieldGroup>

### Accordion Item

`items` 배열의 각 항목:

<ParamFieldGroup>
  <ParamField name="title" type="string" required>
    아코디언 헤더 텍스트
  </ParamField>
  <ParamField name="content" type="string" required>
    아코디언이 펼쳐질 때 표시되는 내용
  </ParamField>
  <ParamField name="icon" type="string">
    표시할 이모지 또는 아이콘 (선택사항)
  </ParamField>
</ParamFieldGroup>
