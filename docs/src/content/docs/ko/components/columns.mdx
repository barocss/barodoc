## 2열

<Columns columns={2}>
  <Column>
    **왼쪽 열**
    
    이 콘텐츠는 왼쪽에 나타납니다. 비교나 관련 콘텐츠에 적합합니다.
  </Column>
  <Column>
    **오른쪽 열**
    
    이 콘텐츠는 오른쪽에 나타납니다. 열은 모바일 장치에서 수직으로 쌓입니다.
  </Column>
</Columns>
