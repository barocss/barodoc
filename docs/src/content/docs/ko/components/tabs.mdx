Tabs는 관련 콘텐츠를 전환 가능한 패널로 정리합니다.

## 패키지 매니저 예시

<Tabs client:load items={[
  { label: "npm", value: "npm", children: "npm install @barodoc/core" },
  { label: "pnpm", value: "pnpm", children: "pnpm add @barodoc/core" },
  { label: "yarn", value: "yarn", children: "yarn add @barodoc/core" }
]} />

```mdx
<Tabs client:load items={[
  { label: "npm", value: "npm", children: "npm install @barodoc/core" },
  { label: "pnpm", value: "pnpm", children: "pnpm add @barodoc/core" },
  { label: "yarn", value: "yarn", children: "yarn add @barodoc/core" }
]} />
```

## 코드 예시

<Tabs client:load items={[
  { 
    label: "JavaScript", 
    value: "js", 
    children: "const greeting = 'Hello, World!';\nconsole.log(greeting);" 
  },
  { 
    label: "TypeScript", 
    value: "ts", 
    children: "const greeting: string = 'Hello, World!';\nconsole.log(greeting);" 
  },
  { 
    label: "Python", 
    value: "py", 
    children: "greeting = 'Hello, World!'\nprint(greeting)" 
  }
]} />

```mdx
<Tabs client:load items={[
  { label: "JavaScript", value: "js", children: "const greeting = 'Hello';" },
  { label: "TypeScript", value: "ts", children: "const greeting: string = 'Hello';" },
  { label: "Python", value: "py", children: "greeting = 'Hello'" }
]} />
```

## 기본값 설정

`defaultValue`로 처음 활성화될 탭을 설정:

<Tabs client:load defaultValue="pnpm" items={[
  { label: "npm", value: "npm", children: "npm install" },
  { label: "pnpm", value: "pnpm", children: "pnpm install" },
  { label: "yarn", value: "yarn", children: "yarn" }
]} />

```mdx
<Tabs client:load defaultValue="pnpm" items={[
  { label: "npm", value: "npm", children: "npm install" },
  { label: "pnpm", value: "pnpm", children: "pnpm install" },
  { label: "yarn", value: "yarn", children: "yarn" }
]} />
```

## Props

<ParamFieldGroup>
  <ParamField name="items" type="TabItem[]" required>
    탭 아이템 배열 (아래 Tab Item 참조)
  </ParamField>
  <ParamField name="defaultValue" type="string">
    처음 활성화될 탭의 value
  </ParamField>
  <ParamField name="client:load" type="directive" required>
    클라이언트 사이드 인터랙티브에 필요한 Astro 디렉티브
  </ParamField>
</ParamFieldGroup>

### Tab Item

`items` 배열의 각 아이템:

<ParamFieldGroup>
  <ParamField name="label" type="string" required>
    사용자에게 표시되는 탭 버튼 텍스트
  </ParamField>
  <ParamField name="value" type="string" required>
    탭의 고유 식별자
  </ParamField>
  <ParamField name="children" type="string" required>
    탭이 활성화될 때 표시되는 콘텐츠
  </ParamField>
</ParamFieldGroup>
