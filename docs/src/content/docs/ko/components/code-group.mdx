코드 그룹을 사용하면 여러 코드 예제를 탭 인터페이스로 표시할 수 있어, 같은 기능을 다른 언어로 보여주기 쉽습니다.

## 기본 사용법

<CodeGroup titles={["JavaScript", "Python", "Go"]}>

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

```python
greeting = "Hello, World!"
print(greeting)
```

```go
package main

import "fmt"

func main() {
    greeting := "Hello, World!"
    fmt.Println(greeting)
}
```

</CodeGroup>
