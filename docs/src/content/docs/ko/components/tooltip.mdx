툴팁은 사용자가 용어나 요소 위에 마우스를 올렸을 때 추가 컨텍스트를 제공합니다.

## 기본 사용법

<Tooltip client:load tip="Application Programming Interface - 프로그램이 서로 통신하는 방법">API</Tooltip> 위에 마우스를 올려 더 많은 정보를 확인하세요.

```mdx
<Tooltip client:load tip="Application Programming Interface">API</Tooltip> 위에 마우스를 올려 더 많은 정보를 확인하세요.
```
