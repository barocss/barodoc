Barodoc은 Astro 기반의 문서화 프레임워크입니다. Mintlify 스타일의 문서 사이트를 쉽게 만들 수 있습니다.

## 주요 기능

- **MDX 지원** - Markdown에 React 컴포넌트를 함께 사용
- **다국어 지원** - 여러 언어로 문서 작성 가능
- **다크 모드** - 자동 다크/라이트 모드 전환
- **검색** - Pagefind 기반 클라이언트 사이드 검색
- **플러그인 시스템** - 기능 확장 가능

## 프로젝트 구조

```
my-docs/
├── barodoc.config.json   # 설정 파일
├── docs/
│   ├── en/               # 영문 문서
│   └── ko/               # 한국어 문서
└── public/
    └── logo.svg          # 로고 이미지
```
